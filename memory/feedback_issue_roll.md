The homepage issue roll (`.heroPosterRoll`) should stay as a vertical column of large Playfair Display text links with left border accents and hover-slide animation.

**Why:** A horizontal flowing/inline approach was tried and reverted immediately — user preferred the original editorial list feel.

**How to apply:** If the issue roll ever comes up for redesign again, propose alternatives verbally first rather than implementing. The vertical column is the locked treatment.
