# Implementation Guides

Store phase-scoped framework guides in this directory.

Use an implementation guide when the main question is:

- what a framework concept means
- how that concept works in the current phase
- how to understand the concept in plain language with tiny examples

Implementation guides teach concepts. They do not own repo edit order. For
that, use `docs/learnings/walkthroughs/`.

For the detailed rules, format, and when-to-use guidance, see
`docs/learnings/syllabus-workflow.md`.
