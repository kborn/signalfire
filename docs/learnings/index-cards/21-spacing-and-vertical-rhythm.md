# Spacing And Vertical Rhythm

Vertical rhythm means using consistent spacing between headers, sections,
paragraphs, and link groups so the page reads as intentional structure instead
of one collapsed block.

Why it matters here:
Phase 6.1 needs minimal shared spacing in the layout and page sections even
before later visual polish work happens.

Rule of thumb:
Use a few consistent gaps and padding values across the shared shell before
adding detailed styling. The goal is readable structure, not polished design.
