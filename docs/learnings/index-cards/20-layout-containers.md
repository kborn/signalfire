# Layout Containers

A container is a shared wrapper that limits content width, centers the page,
and adds horizontal padding so content does not touch the viewport edge.

Why it matters here:
Phase 6.1 does not need polished design, but the shared layout should still
give pages readable width and consistent outer structure.

Rule of thumb:
If every page should sit inside the same readable content column, that belongs
in the shared layout/container layer rather than repeated in each page.
