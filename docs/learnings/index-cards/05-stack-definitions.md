# Learning Card 05: Stack Definitions

- `Node.js` / `Node` = JS runtime
- `JavaScript` = language
- `TypeScript` = JS + types
- `React` = UI library
- `Next.js` = web framework on React
- `NestJS` = backend framework on Node
- `Postgres` = database
- `Prisma` = ORM / DB client
- `pnpm` = package manager
- `Turbo` = monorepo task runner
- `ESLint` = linter
- `Prettier` = formatter
- `Husky` = git hook runner

Repo map:

- `apps/web` = Next app
- `apps/api` = Nest app
- `apps/api/prisma` = Prisma schema/migrations
