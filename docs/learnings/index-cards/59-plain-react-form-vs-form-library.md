# Plain React Form Vs Form Library

Start with normal inputs plus React state unless a library removes real
complexity you already understand.

Why it matters here:

Phase 10.4 is a good place to learn the mechanics directly: values, errors,
submit handling, and success/failure behavior. A library can hide those basics
before you know what problem it is solving.

Rule of thumb:

- use plain React first when the form is still understandable
- consider a library later if the manual code becomes repetitive or hard to
  reason about across multiple forms
